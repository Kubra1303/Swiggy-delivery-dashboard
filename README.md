# Swiggy Delivery Performance Dashboard

## Project Overview
This project is an interactive Power BI dashboard created to analyze Swiggy restaurant delivery performance across different cities and areas.

## Data Source
Dataset sourced from Kaggle:Swiggy Restaurant Dataset

## The dashboard provides insights into:
- Average Delivery Time
- Average Ratings
- Total Number of Restaurants
- Average Price
- Food Type Distribution
- Top Cities with Most Restaurants
- Top Areas with Most Restaurants

## Tools Used
- Power BI
- DAX Measures
- Data Visualization
- Data Cleaning & Transformation

## Key KPIs
- Average Delivery Time
- Average Ratings
- Total Restaurants
- Average Price

## Dashboard Features
- Interactive slicers for City and Area
- Donut chart for Food Type Distribution
- Map visualization for city-wise restaurant analysis
- Bar chart showing top restaurant areas
- Dynamic insight summary

## Files Included
- Swiggy_Dashboard.pbix
- dashboard_screenshot.png
- swiggy_data.csv

## Project Objective
To analyze restaurant performance and delivery efficiency, helping identify trends in pricing, ratings, and food distribution.
